## CI/CD Pipeline Flow
This project uses **GitHub Actions** to automate the workflow:
1. **Push:** Any change pushed to the `main` branch triggers the pipeline.
2. **Build & Check:** The system verifies that `index.html`, `style.css`, and `script.js` exist.
3. **Deploy:** The site is automatically updated and hosted on **GitHub Pages**.
