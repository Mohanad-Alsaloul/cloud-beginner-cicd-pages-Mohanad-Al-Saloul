What is CI/CD?

CI/CD is a cloud practice that automates building, testing, and deploying applications. Continuous Integration (CI) checks code automatically after every push. Continuous Deployment (CD) publishes updates automatically. This reduces errors and saves time.

My Pipeline

When I push code to GitHub, GitHub Actions runs automatically. It checks required files and then deploys the site to GitHub Pages.
